== Installation ==

**Installation through WP Admin**

1. Go to your WordPress Admin > Plugins and upload the zip file
2. Click Activate

**Installation through FTP**

1. Unzip the zip file
2. Upload the folder to /wp-content/plugins/
3. In your WP Admin, go to Plugins and activate the plugin

A note for Multisites: You can enable the plugin network wide or within a site, both will work.
